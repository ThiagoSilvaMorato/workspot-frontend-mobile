# Arquitetura

Antes de implementar qualquer funcionalidade:

Descubra:

- organização das pastas
- padrão das features
- padrão dos serviços
- padrão das páginas
- padrão dos componentes

Nunca assumir uma arquitetura.

Sempre descobrir primeiro.

Caso existam múltiplos padrões, seguir o predominante.
