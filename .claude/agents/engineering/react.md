# React

Antes de criar qualquer componente:

Procure por:

- componente semelhante
- hook semelhante
- provider semelhante
- contexto semelhante

Sempre:

- usar TypeScript
- manter componentes pequenos
- manter hooks pequenos
- reutilizar lógica

Evitar:

- prop drilling
- useEffect desnecessário
- estados duplicados
- componentes gigantes

Nunca criar componentes duplicados.
